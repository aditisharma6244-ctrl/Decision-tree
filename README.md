This project implements a simple deterministic decision tree using Python.
A deterministic decision tree gives the same output for the same input every time.  
This implementation uses a basic recursive approach.
# FEATURES:
- No randomness
- Simple tree building logic
- Easy to understand
- Works on categorical data
# How It Works:
- Selects the first feature
- Splits data based on values
- Recursively builds tree
- Predicts output using tree traversal
# RUNNING CODE ROM:
python decision_tree.py
