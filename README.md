
# Assignment Purpose:
This project is created as part of a role-simulation assignment implements a simple deterministic decision tree using Python.

It implements a deterministic decision tree that models decision-making logic.
A deterministic decision tree gives the same output for the same input every time.  
This implementation uses a basic recursive approach.
# FEATURES:
- No randomness
- Simple tree building logic
- Easy to understand
- Works on categorical data
# How It Works:
- Selects the first feature
- Splits data based on values
- Recursively builds tree
- Predicts output using tree traversal
# Example:
Is it raining ?
   - Yes - stay home
   - No - Is it sunny?
      - yes -  Go for picnic
      -  No - Is it windy?
         - Yes - Go with friends
         - No - Go for walk
# RUNNING CODE ROM:
python decision_tree.py
